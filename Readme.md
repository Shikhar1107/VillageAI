# Village AI – Voice-Based Agricultural Advisory System

## Overview
Village AI is a voice-first agricultural advisory prototype designed for rural farmers using feature phones. The system processes spoken crop queries and delivers safe advisory responses.

## Current Status
- Architecture defined
- AWS services identified
- Backend & frontend structure initialized
- Development in progress (Prototype Phase)

## Tech Stack
- FastAPI (Backend)
- React (Frontend)
- AWS EC2
- AWS Transcribe
- AWS Polly
- Amazon S3

## Roadmap (7-Day Prototype Plan)
- Day 1: Backend & EC2 Setup
- Day 2: STT Integration
- Day 3: Intent + Knowledge
- Day 4: Safety Layer
- Day 5: TTS Integration
- Day 6: Deployment & Testing
