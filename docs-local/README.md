# Local Documentation

This folder contains local-only documentation that is not tracked in git.

## Structure

- **presentations/** - Presentation materials, slides, and generation guides
- **backend-notes/** - Internal backend documentation, API notes, and development guides
- **meeting-notes/** - Team meeting notes and discussions
- **planning/** - Internal planning documents, brainstorming, and drafts

## Important

This entire `docs-local/` folder is gitignored. Files here:
- Will NOT be committed to the repository
- Are for local/team internal use only
- Should contain sensitive or draft materials

For public documentation, use the main `docs/` folder or `.kiro/specs/` directory.
